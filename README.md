# Student Insight Generator

Student Insight Generator is a simple Python project that analyzes student marks and provides meaningful insights about overall class performance.

This project focuses on clarity, logic, and clean data handling using basic Python concepts.

---

## Features

- Input student names and marks
- Display individual student details
- Assign grades (A, B, C, F)
- Count pass and fail students
- Find highest and lowest marks
- Calculate average score
- Identify class performance level
- Display topper details

---

## Technologies Used

- Python
- Lists
- Dictionaries
- Loops
- Conditional Statements

---

## How It Works

1. User enters the number of students.
2. For each student, name and mark are collected.
3. The program stores data using dictionaries inside a list.
4. Student performance is analyzed using basic logic.
5. Final insights are displayed in a clean and readable format.
